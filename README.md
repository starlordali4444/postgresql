# SQL Mastery — PostgreSQL Edition

A comprehensive 6-week, 29-session SQL course built on **PostgreSQL 18** with a real-world Indian retail dataset (RetailMart V3).

## 🔗 Live Site

**https://starlordali4444.github.io/postgresql/**

Past cohorts (batches 21–26) live in the public archive: **https://starlordali4444.github.io/postgresql-legacy/**

## What's Inside

- **30 lecture decks** (PDF) — fundamentals through the analytics capstone
- **11,200 practice problems** — 28 topics × 4 difficulty tiers (easy / medium / hard / crazy) × 100 each
- **RetailMart V3 dataset** — 16 schemas, 55 tables, ~2.7M rows
- **Capstone project** — build an enterprise analytics layer (76 views, 13 materialized views, 30 indexes) and ship it as a live dashboard
- **Setup guides** (macOS + Windows) and a one-click GitHub Codespace

Every page offers **per-file downloads** and **"Download all" ZIP bundles** (in `downloads/`); the full dataset is a GitHub Release asset.

## Access

Notes, practice, datasets, and curriculum are unlocked with a **batch access code** (shared by the instructor with each cohort). Don't have one? Use the **Request Access** form on the site (WhatsApp or email).

## Repository Structure (flat)

```
index.html        -> Landing page
notes/            -> 30 topic decks (PDF) + index
practice/         -> retailmart_v3/<topic>/<topic>_{easy,medium,hard,crazy}.sql + index
datasets/         -> v3/ (CSV + setup SQL) + browsable index
curriculum/       -> SQL_Curriculum_v4.md (current spec)
setup/            -> macOS + Windows install guides
downloads/        -> section ZIP bundles (notes, practice, curriculum, project)
project/          -> capstone dashboard preview
viewer.html       -> in-browser file viewer (SQL / Markdown)
batch_notes/      -> per-batch Excel + schedule PDF
```

## About the Dataset

**RetailMart V3** — a realistic Indian e-commerce database:
- 16 schemas, 55 tables, ~2.7M rows
- Full business lifecycle: Supplier → Warehouse → Store → Customer
- Built-in data-quality scenarios for real-world practice

## Tools

- **PostgreSQL 18** — database engine
- **pgAdmin 4** — admin interface
- **VS Code** — query editor
