# SQL Mastery Course - PostgreSQL Edition

A comprehensive 6-week, 29-session SQL course built on **PostgreSQL 18** with a real-world retail dataset.

## Repository Structure

```
site/               -> Course website content
  --- setup/        -> Installation & setup guides (free)
  --- schema/       -> Database schema reference (free)
  --- weeks/        -> Weekly exercises (gated)
  --- datasets/     -> CSV & SQL downloads (gated)
  --- archive/      -> Past batch content (gated)
```

## About the Dataset

**RetailMart V3** - a realistic Indian retail database:
- **16 schemas, 55 tables, ~2.7M rows**
- Full business lifecycle: Supplier -> Warehouse -> Store -> Customer
- Built-in data quality issues for real-world practice
- Covers every major SQL concept from basics to advanced analytics

## Course Outline

| Week | Topics |
|------|--------|
| **1** | SQL Basics, DDL, Data Types, Constraints |
| **2** | Filtering, Scalar Functions, Aggregates, Conditional Logic |
| **3** | Joins (all types), Set Operations, Subqueries, CTEs |
| **4** | Window Functions (3 parts) + Practice Day |
| **5** | Query Performance, Indexing, Transactions, Normalization |
| **6** | Views, Materialized Views, Functions, Stored Procedures, Advanced Topics |

## Tools
