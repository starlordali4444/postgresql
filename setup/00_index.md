#  Day 1 - Environment Setup

Welcome to **Day 1**! Before writing any SQL, let's get your tools installed.

###  Pick Your OS

| Your OS     | Guide                                  |
| :---------- | :------------------------------------- |
| **macOS**   | [setup_mac.md](./setup_mac.md)         |
| **Windows** | [setup_windows.md](./setup_windows.md) |

###  What Gets Installed

| Tool           | Purpose                                                   |
| :------------- | :-------------------------------------------------------- |
| **PostgreSQL** | The database engine - where your data lives               |
| **pgAdmin 4**  | Visual database manager - browse tables, run queries      |
| **VS Code**    | Code editor - write `.sql` files with syntax highlighting |
| **Git**        | Version control - track your work and collaborate         |

>  **Mac users:** You can run `bash install_mac.sh` to install everything automatically!

---

**Next Step ->** After installation, open `query.sql` to run your first commands!
